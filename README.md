# replicater-site

Source for [replicater.us](https://replicater.us/) — a single-page static site (Monet-inspired palette, "Microbe — 1%").

Deployed to Vercel under the `microbe-impressionist` project.

## Structure

- `index.html` — the whole site (inlined CSS + JS)
- `favicon.svg`
- `paper.pdf` — research paper, linked from the top **Research** button. Drop the file in the repo root with this exact filename and the link will work automatically.
